압축 방식

이 브라우저는 WebP 저장을 지원하지 않아 JPEG로 대체돼요.

메타데이터

설정을 바꾸면 목록의 이미지가 자동으로 다시 압축돼요. 기본값은 EXIF(위치정보 등) 제거 — ‘메타데이터 유지하기’를 켜면 원본 EXIF를 결과 파일(WebP·JPEG·PNG)에 복사해요.

    배경 제거

    테두리에서 배경색을 학습해 자동으로 지워요 — 단색·그라데이션 배경에 잘 맞아요. 자동 결과가 아쉬우면 브러시로 직접 다듬으세요.

    Base64 → 이미지

    Data URI 또는 순수 Base64 문자열을 붙여넣으세요. 이미지 형식은 자동으로 알아내요.

    메타데이터 보기·편집

    픽셀은 건드리지 않고 메타데이터만 읽고 고쳐요 — 화질 손실 0. JPEG는 EXIF, PNG는 텍스트 청크를 사용해요.

    카메라 — 선택하면 실제 기종의 EXIF 표기 형식으로 채워져요
    방향 (Orientation) — 픽셀은 그대로, 뷰어의 표시 회전만 바뀌어요
    소프트웨어

    PNG에는 카메라·방향·GPS 항목이 없어 저장되지 않아요.

    빈 칸(또는 ‘태그 없음’)으로 두면 해당 항목은 저장되지 않아요 — 기존 값도 삭제돼요.

    🔒 모든 변환은 내 브라우저 안에서만 일어나요 — 어떤 이미지도 서버로 전송되지 않아요. 인터넷이 끊겨도 동작해요.
