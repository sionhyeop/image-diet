`shell/svgtool.py`에 `rasterize` 추가. **기존 `vectorize`가 내부적으로 계산하는 "색별 단순화 루프"를 재사용**한다. `vectorize`가 `_compute_entries(work, opts)` 같은 형태로 분리돼 있지 않다면, 그 계산부(전처리 축소 → 양자화 인덱스/팔레트 → 색마다 마스크 → `_trace_loops` → `_simplify_loop`(tol) → minArea 필터)를 `_compute_entries(img, opts) -> (entries, w, h)` 헬퍼로 **추출**하고 `vectorize`와 `rasterize` 둘 다 그것을 호출하게 리팩터한다. `entries`는 `[(rgb_tuple, [loop_points, ...]), ...]`(팔레트 순서) 형태.

```python
from PIL import Image, ImageDraw  # 파일 상단 import에 ImageDraw 추가


def rasterize(img, opts, box=None):
    entries, w, h = _compute_entries(img, opts)   # vectorize와 공유하는 계산부
    canvas = Image.new("RGB", (w, h), (255, 255, 255))
    draw = ImageDraw.Draw(canvas)
    for rgb, loops in entries:
        for loop in loops:
            if len(loop) >= 3:
                draw.polygon([(float(x), float(y)) for x, y in loop], fill=rgb)
    if box:
        canvas.thumbnail(box, Image.LANCZOS)
    return canvas
```

주의: `vectorize`의 반환(SVG 문자열)과 기존 4개 테스트는 **그대로 통과**해야 한다(리팩터가 동작을 바꾸면 안 됨). `_compute_entries` 추출 후 `vectorize`는 그 entries로 SVG 문자열을 만들도록 유지.

    def _schedule(self):

            try:
                self.after_cancel(self._pending)
            except Exception:
                pass
        self._pending = self.after(400, self._render_preview)

    def _render_preview(self):
        self._pending = None
        if not self.files:

        opts = svgtool.opts_from_controls(
            self.vars["colors"].get(), self.vars["detail"].get(),
            self.vars["simplify"].get(), self.vars["noise"].get(),
            self.vars["gap"].get(), self.smooth.get())
        src = self.files[0]
        threading.Thread(target=self._render_work, args=(src, opts), daemon=True).start()

    def _render_work(self, src, opts):
        try:
            from PIL import Image
            img = Image.open(src).convert("RGB")
            preview_img = svgtool.rasterize(img, opts, box=(260, 220))
        except Exception:
            return
        self._post(lambda: self.preview.set(preview_img))

    def _post(self, fn):
        try:
            if self.winfo_exists():
                self.after(0, fn)
